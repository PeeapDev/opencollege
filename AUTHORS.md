OpenCollege welcomes contributions from the global education-technology
community. See `CONTRIBUTING.md` for how to get involved.

All contributors are listed in the project's Git history and on the GitHub
contributors page:
https://github.com/PeeapDev/opencollege/graphs/contributors

## Governance

OpenCollege is developed as an open-source Digital Public Good under the
MIT license (see `LICENSE`). Strategic direction is set by the maintainers
above, with input from partner institutions using the platform in production.

Major architectural decisions are documented in pull requests and releases.
Security-sensitive decisions follow the process described in `SECURITY.md`.
